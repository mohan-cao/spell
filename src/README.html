VoxSpell - Spelling for Professionals

Mohan Cao (mcao024)

Installation instructions:

  1. Unzip the file into any directory you wish
  2. Open the exec.sh file. Set up is automatic.

Features:

First time run:

Choose levels to unlock up to. This will be for the default list only.

Main menu:

  • The button "New game" leads to a level select menu where you can select a level to initiate a quiz.
  • The button "Review" to go to level menu to review previously failed words
  • The button "Statistics" to check statistics of words.
  • The button "Settings" to mute background music and to set word lists (must be in format of default list), and to change default voice (you can always change voice later in quiz).

Dynamic background fetching from an open-source image API is possible with an internet connection only.

Level menu:

Select a level to make a game. Hovering over levels currently displays MASTERY of words in that level, but will in future display your personal best highscore.

Quiz game:

Enter words in the text area, then confirm your word with the ENTER key or by pressing the Confirm button. You cannot type nothing, which will give you a warning, or type excessively much, but there is no limit to number of characters typed.

Currently, all letters in a-z, A-Z, ' (apostrophe), _ (space), and - (hyphen) are allowed.

You are given your high score, personal best, and maximum possible score on the top. Currently, beating your personal best does not give a reward.

Hover over the icons on the right to get a tooltip of what they do. (They are buttons for changing the voice, and repeating the word)

You can also get the word definition (actually a word example) with the "Get word definition" button. This requires an active internet connection to work, as it is dynamically fetched from an API.

Statistics:

You can see a barchart of overall word statistics, and a table of all words completed.

You can filter by level or word, and the table gives you a detailed description of what words you are most comfortable with and those that you struggle with by simply pressing the headings to order the table headings.

You can also clear statistics using the 'Clear Stats' button.

Settings:

Buttons:

  • Mute background music
  • Set word list
    • All lists that have been loaded are initialised with unlocked level 1 only.
    • There is no option to load files of a different format to that of the NZCER spelling list.
    • Don't worry about moving word lists around accidentally: as long as you can relocate the same list with the file chooser, you can play the same file. Coincidentally, this means that two different spelling lists with the same name will have their stats merged.
  • Set default voice
    • Currently, the voices are not very human-readable. Still, a list of possible voices can be generated (not sure if all of them work though, but the default works and the nz voice works)
